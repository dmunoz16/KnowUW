About KnowUW


KnowUW is an informational website created by Information Technology majors for the class TINFO 230.

We decided to create a website were people could use the interactive map to see pictures and information about each bulding
on campus while also have access to important events and popular links related to UWT all available in one place.



Website Contributors: Benjamin Wheat, Diana Muñoz, Kornsavanh Sisa-at, Ben Newland
